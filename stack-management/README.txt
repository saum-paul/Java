1. The Application uses Maven as a build and run tool.

2. Application uses JUnit and Log4j as utility frameworks.

3. To build the application -  mvn clean package -DskipTests

4. To run the application -  mvn exec:java

5. The run without maven:
	* java Client
 	* java Client 2019-11-14T09:00:00
 	* java Client 2019-11-14T09:00:00 input.txt
 	
 6. Default arguments are configured in Maven and also in the client.
